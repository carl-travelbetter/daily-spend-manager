Travel Spend Manager TL:DR

Keep things simple—follow the TL;DR if you’re in a hurry.

TL;DR

1. Start New → Set a budget → Select Your Trip Dates → Click Calculate.

2. Add expenses as you go.

3. Monitor your balance.

What you need to get started

  • Webrowser - e.g. Chrome, IE, Safari
  • Your Holiday Spending Budget
  • The Dates of your Trip

Useful info

  • All changes saved locally
  • No account required
  • Works in any web browser
